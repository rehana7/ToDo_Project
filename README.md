# ToDo_Project 
This project is generated using Vue JS.

## Environment Setup 
Need to install the latest version of nodejs  
```PowerShell
node --version
```

To get the starter code : Open the empty folder in VS Code and in 
```Terminal
npm create vite@latest . -- --template vue
```
```Terminal
npm install
```
```Terminal
npm run dev
```
